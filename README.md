# Genie of a Thousand Nights

## How to Play
The Genie moves automatically. You control the **Night** (the obstacles).
- **Goal**: Align the gaps with the Genie's path.
- **Controls**:
  - **PC**: UP / DOWN Arrow keys (or W / S).
  - **Touch**: Tap & Hold TOP of screen (Move Up) / BOTTOM of screen (Move Down).

## Features
- Procedural Obstacles
- Lantern Collection (Bronze, Silver, Gold, Cursed)
- Difficulty Scaling
- Dynamic Particle Background

## Installation
Just open `index.html` in your favorite web browser.
